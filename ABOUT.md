# BODY PALETTE 展示会抽選パック

## 一言で言うと

BODY PALETTEの展示会ブースで、来場者にQRコードや共有URLから「BODY PALETTE ギフトパック」を開封してもらう静的MVPです。足を止めるきっかけを作り、そのままスタッフが会話に入りやすくするためのフックとして使います。

## 何ができるのか

- URLパラメータ `type` に応じて、通常来場者／相談希望／デモ後／セミナー参加／VIPの表示を出し分け
- 来場者画面では、セッション体験、プロテイン、簡易版健康動態モニタリングなどの特典を何度でも続けて抽選可能
- 特典コードと有効期限を来場者画面に表示
- 管理画面で発行済み／使用済み特典を確認し、使用済み・未使用の切り替えが可能
- 管理画面で特典ごとの当選確率を％で設定可能
- `type` / `code` / `lead` のURLパラメータに応じて、特定の来場者だけ当選確率を上げたり、特典を確定させたりできる
- 参考ルーレットサイトと同じ、白〜薄ピンク背景・ピンク見出し・横長ロゴ・淡いブランドカラーの展示会向けデザイン

現在の抽選特典は以下の5つです。

- オンサイトセッションプレゼント
- お姫様抱っこ
- プロテインプレゼント
- オンラインセッション体験チケットプレゼント
- 簡易版健康動態モニタリング プレゼント

## 構成

- `index.html` — 来場者用のパック開封画面
- `admin.html` — スタッフ用の管理画面
- `styles.css` — BodyPalette風の画面デザイン
- `coupon-core.js` — 抽選データ、確率設定、特別ルール、ローカル保存処理
- `app.js` — 来場者画面の開封演出
- `admin.js` — 管理画面の表示、フィルター、使用済み切り替え、抽選設定編集
- `assets/` — ロゴ、背景、サービス画像などの表示素材

## 使い方

```bash
cd /Users/hayatokagami/⭐FW/bodypalette-coupon-pack
python3 -m http.server 4174
```

```text
http://localhost:4174/?type=general
http://localhost:4174/admin.html
```

特別ルールを使う場合は、URLに条件を追加します。

```text
http://localhost:4174/?type=vip
http://localhost:4174/?type=demo&lead=hot
http://localhost:4174/?type=general&code=demo-vip
```
